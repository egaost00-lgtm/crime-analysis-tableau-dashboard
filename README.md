# 🚔 Crime Analysis Dashboard using Tableau

## 📌 Project Overview
This project is an interactive Tableau dashboard created to analyze crime data and identify patterns, trends, and insights through data visualization.

## 🎯 Objectives
- Analyze crime trends over time
- Identify high-crime locations
- Compare crime categories
- Build an interactive dashboard for easy analysis

## 🛠️ Tools Used
- Tableau
- Microsoft Excel
- Data Visualization

## 📊 Dashboard Features
- Interactive Filters
- KPI Cards
- Trend Analysis
- Crime Category Analysis
- Location-Based Analysis
- Dynamic Charts

## 📈 Key Insights
- Identified crime hotspots.
- Analyzed crime trends over time.
- Compared different crime categories.
- Created an easy-to-use interactive dashboard.
